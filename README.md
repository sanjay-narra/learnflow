# 🚀 LearnFlow – Learning Platform UI

**LearnFlow** is a modern learning platform interface built using **HTML, CSS, and JavaScript**.
It provides a clean and simple UI where users can explore courses and sign up to start learning.

This project demonstrates **frontend development fundamentals**, including page structure, styling, and interactive behavior.
